About the Project

Our project analyzes cycling accidents in France to understand the relationship between cycling infrastructure and safety. Using comprehensive data from official sources, we visualize patterns and trends to identify how infrastructure development impacts accident rates and severity.

Data Collection

Our analysis uses official accident data from the French government's road safety database (BAAC - Bulletin d'Analyse des Accidents Corporels). This database provides detailed information about each accident, including location, severity, circumstances, and infrastructure conditions.

Technologies Used

This visualization platform employs modern web technologies including D3.js for dynamic data visualization, PapaParse for CSV data processing, and JavaScript (ES6+) for data analysis.
